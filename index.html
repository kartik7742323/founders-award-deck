A live product. Thousands of calls a day.

!

!

What I Built

Three Claude-built dashboards. One per Mio Voice workstream.

Connected each team's existing Google Sheets to live dashboards via the Sheets API — no workflow disruption. Three lenses on the same product: how it's adopted, how it onboards, and how it performs.

Open Dashboards
Dashboard 01

Mio Voice Adoption

Total calls, connection rate, avg duration, adoption trends per client. Surfaced the wrong-ROI methodology — and pointed to the need for Call Retry (Woxsen 58.79% vs DY Patil 35%).

Dashboard 02

Mio Voice Onboarding

Tracks each client's onboarding milestones — from sign-up → DID provisioning → first live campaign. Centralized visibility across rolling client launches.

Dashboard 03

Mio AI Voice QC

Categorizes daily call issues by category + sub-category, tracks resolved/unresolved, ranks burning issues by frequency. Drove avg call duration: 27.7s → 31.27s.

Impact · Part 1

Two numbers surfaced — two things we worked on.

Call Connection Rate
~30%
70 of every 100 calls were going unanswered — and unanswered leads were being dropped after a single attempt.
What we're doing about it
1
Call Retry on Mio Voice · IN DEV

Configurable retry attempts (2 / 3 / 4) + time gaps. Projected lift: +10–20% connection rate.

2
Truecaller whitelisting + DID hygiene

QC dials every client's DID — if flagged as spam, we alert Client Success and rotate the agent's DID number. Verified Truecaller IDs lift pickup rates.

Impact · Part 2

The data was misleading. We were down-selling ourselves.

Clients without retry manually relaunched campaigns, dialing the same lead 3–4 times. Every dial counted separately — total dials inflated, connection rate stuck. The dashboard was making Mio look worse than it was.

Old Methodology · Call-level
~30%
Counted every dial separately — even for the same lead.
New Methodology · Lead-level
~56%
A lead is "connected" if they picked up at least once. Same product — finally measured correctly.
From Insight → Client-Facing Release

We revamped & shipped the
Mio AI Insights Dashboard.

Rolled out to every live client on Mio Voice — with the new lead-level methodology baked in. Clients now see Unique Contacts, Avg Attempts / Contact, and Goal Achieved — the number that justifies renewal.

Impact · Part 3

Three more levers — each born from the same data.

Lever 01

Call Connectivity Heat Map → Best-Hour Recommendations

Visualizes pickup rates by hour × day of week, per client's own history. Mio will start recommending the best hours to schedule campaigns — higher connection rate = more calls picked = more revenue.

Zero-cost performance lever
Lever 02

DID Number Health Checks

Identified that spam-flagged or "loan recovery"-labeled numbers were silently killing connection rates. Embedded a manual DID test into every QC review — flagged numbers loop in Client Success immediately.

Embedded into QC workflow
Lever 03

Truecaller Verification

Recommended verified business identity for low-pickup clients. Turns caller identity from liability into a trust signal — more pickups → more calls → more pulse revenue.

Data-driven client advisory

Every lever above was a direct consequence of the dashboard surfacing the right question — and the discipline to follow the data wherever it led.
